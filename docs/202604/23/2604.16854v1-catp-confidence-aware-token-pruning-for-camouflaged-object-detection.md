## 摘要
伪装目标检测（COD）旨在分割与其复杂环境具有极高纹理和结构相似性的目标。利用其长程依赖建模能力，基于 Transformer 的检测器已成为主流方法并达到了最先进（SoTA）的准确率，但其巨大的计算开销严重限制了实际部署。为解决这一问题，我们提出了一种专为 COD 定制的层级置信度感知令牌剪枝框架（CATP）。我们的方法层级地识别并丢弃来自背景和目标内部的易于区分的令牌，将计算集中在关键的边界令牌上。为了补偿剪枝带来的信息损失，我们引入了一种双路径特征补偿机制，将剪枝令牌中的上下文知识聚合到增强特征中。在多个 COD 基准测试上的广泛实验表明，我们的方法在保持高准确率的同时显著降低了计算复杂度，为 COD 模型在现实场景中的高效部署提供了一个极具前景的研究方向。代码将开源。

## Abstract
Camouflaged Object Detection (COD) aims to segment targets that share extreme textural and structural similarities with their complex environments. Leveraging their capacity for long-range dependency modeling, Transformer-based detectors have become the mainstream approach and achieve state-of-the-art (SoTA) accuracy, yet their substantial computational overhead severely limits practical deployment. To address this, we propose a hierarchical Confidence-Aware Token Pruning framework (CATP) tailored for COD. Our approach hierarchically identifies and discards easily distinguishable tokens from both background and object interiors, focusing computations on critical boundary tokens. To compensate for information loss from pruning, we introduce a dual-path feature compensation mechanism that aggregates contextual knowledge from pruned tokens into enriched features. Extensive experiments on multiple COD benchmarks demonstrate that our method significantly reduces computational complexity while maintaining high accuracy, offering a promising research direction for the efficient deployment of COD models in real-world scenarios. The code will be released.

### 1. 核心问题与研究动机
*   **研究背景**：伪装目标检测（COD）要求识别与背景极度相似的目标。目前基于 Transformer 的模型因其长程依赖建模能力成为主流（SoTA），但其自注意力机制的二次计算复杂度导致了巨大的计算开销，限制了在无人机、移动设备等实时场景中的部署。
*   **核心问题**：如何在保证密集预测任务（如分割）精度不受损的前提下，有效减少 Transformer 模型的令牌（Token）数量，从而降低计算量并提升推理速度？
*   **动机**：作者观察到，在 COD 任务中，目标内部和背景深处的 Token 通常易于区分，而真正的挑战在于**目标边界**。因此，可以剪枝掉高置信度的区域，将计算资源集中在模糊的边界区域。

### 2. 方法论
CATP 框架主要由三个核心组件构成：
*   **层级置信度感知剪枝（Hierarchical Confidence-Aware Pruning）**：
    *   在 Transformer 的不同阶段插入轻量级评分头（Scoring Head），预测每个 Token 属于伪装目标的概率（即置信度）。
    *   **双阈值策略**：设定下阈值 $\theta_d$ 和上阈值 $\theta_u$。置信度极低（确定是背景）或极高（确定是目标内部）的 Token 被视为“简单”并进行剪枝；置信度在两者之间的 Token 被视为“模糊/困难”并予以保留，进入深层网络进一步精炼。
*   **双路径特征补偿（Dual-Path Feature Compensation, DPFC）**：
    *   为了防止直接丢弃 Token 导致的信息丢失，该机制将剪枝掉的背景和前景 Token 分别聚合成两个紧凑的**原型 Token**（Prototype Tokens）。
    *   **概率加权聚合**：靠近决策边界的 Token 获得更高权重，因为它们包含更多潜在的判别信息。这两个原型 Token 会被重新拼接回序列中，参与后续计算。
*   **层级令牌回填（Hierarchical Token Refilling）**：
    *   为了适配解码器所需的密集预测（全分辨率特征），采用递归回填策略，将深层精炼后的稀疏特征插回原始空间位置，确保空间结构的一致性。

*   **高效性**：在 CFRN 模型上，CATP 减少了约 **38.8% 的 GFLOPs**，并将 FPS 提升了 **34.8%**，而精度损失微乎其微（部分指标甚至有所提升）。

### 7. 优点与亮点
*   **任务定制化**：不同于传统的图像分类剪枝，CATP 专门针对 COD 的“边界敏感性”设计了双阈值剪枝逻辑。
*   **信息无损化尝试**：通过双路径原型聚合，在大幅压缩序列长度的同时，保留了全局上下文的完整性。
*   **即插即用**：该框架具有良好的通用性，可以轻松集成到不同的基于 Transformer 的 COD 模型中。
