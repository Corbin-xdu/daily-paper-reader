## 摘要
长视频理解对视觉语言模型（VLM）而言具有内在挑战性，因为其帧数庞大。由于每个视频帧通常会扩展为数十或数百个 token，大语言模型（LLM）有限的上下文长度迫使 VLM 只能稀疏地感知视频帧，从而导致时序信息的丢失。为解决这一问题，我们探索了在 LLM 最终层实现“每帧一个 token”的极致视频 token 压缩。我们的核心见解是，先前方法广泛采用的基于启发式的压缩容易导致信息丢失，因此有必要将 LLM 层监督为用于“token 级压缩”的可学习且渐进式模块（LP-Comp）。这种压缩使我们的 VLM 能够处理 2 到 4 倍的视频帧，并提升性能。为了进一步提高 token 效率，我们研究了“帧级压缩”，即通过 LLM 层的内部注意力分数选择与查询最相关的帧，称之为“问题条件压缩”（QC-Comp）。与以往研究的一个显著区别在于，我们通过将长视频分割成短片段并采用局部注意力，缓解了长上下文中 LLM 注意力的位置偏差（即过度集中在序列开头和结尾的问题）。总之，我们结合了 token 级和帧级压缩，构建了一个名为 \name 的长视频理解极致压缩模型，实现了显著更高的压缩率并支持更密集的帧采样。我们的 \name 模型基于 VideoChat-Flash 进行微调，采用了一种数据高效的“监督压缩微调”阶段，仅需 2.5% 的监督微调数据，便将 LVBench 上的准确率从 42.9% 提升至 46.2%，并增强了多个其他长视频基准测试的表现。

## Abstract
Long video understanding is inherently challenging for vision-language models (VLMs) because of the extensive number of frames. With each video frame typically expanding into tens or hundreds of tokens, the limited context length of large language models (LLMs) forces the VLMs to perceive the frames sparsely and lose temporal information. To address this, we explore extreme video token compression towards \emph{one token per frame} at the final LLM layer. Our key insight is that heuristic-based compression, widely adopted by previous methods, is prone to information loss, and this necessitates supervising LLM layers into \emph{learnable} and \emph{progressive} modules for \emph{token-level compression} (LP-Comp). Such compression enables our VLM to digest 2x-4x more frames with improved performance. To further increase the token efficiency, we investigate \emph{frame-level compression}, which selects the frames most relevant to the queries via the internal attention scores of the LLM layers, named \emph{question-conditioned compression} (QC-Comp). As a notable distinction from previous studies, we mitigate the position bias of LLM attention in long contexts, \emph{i.e.}, the over-concentration on the beginning and end of a sequence, by splitting long videos into short segments and employing local attention. Collectively, our combined \emph{token-level} and \emph{frame-level} leads to an e\textbf{x}treme compression model for long video understanding, named \textbf{\name}, achieving a significantly larger compression ratio and enabling denser frame sampling. Our \name is finetuned from VideoChat-Flash with a data-efficient \emph{supervised compression tuning} stage that only requires 2.5\% of the supervised fine-tuning data, yet boosts the accuracy from 42.9\% to 46.2\% on LVBench and enhances multiple other long video benchmarks.

### 1. 核心问题与整体含义（研究动机和背景）
长视频理解的主要瓶颈在于**视觉 Token 爆炸**。现有的视觉语言模型（VLM）在处理每一帧时会生成数十甚至数百个 Token，这迅速填满了大语言模型（LLM）的上下文窗口，迫使模型不得不进行稀疏采样（如每秒仅取极少帧），从而丢失了关键的时序细节。
论文的核心动机是探索**极致压缩**：能否在 LLM 的最终层将每一帧压缩至仅剩 **1 个 Token**，同时通过智能筛选保留最相关的帧，从而在不增加计算负担的前提下，允许模型“看”到更密集的视频帧。

### 2. 方法论：XComp 架构
XComp 结合了两个维度的压缩策略：

*   **LP-Comp（可学习且渐进式 Token 级压缩）：**
    *   **核心思想：** 放弃传统的基于启发式（如简单池化或随机丢弃）的压缩，转而通过监督学习让 LLM 层学会如何将一帧的多个 Token 浓缩。
    *   **技术细节：** 采用**后缀保留（Suffix-preservation）**策略，利用解码器 LLM 的因果注意力特性，让后面的 Token 吸收前面的信息。
    *   **渐进式调度：** 使用**余弦调度函数**，在 LLM 的 $L$ 层中平滑地减少每帧的 Token 数，直到最后一层达到每帧 1 个 Token。
*   **QC-Comp（问题引导的帧级压缩）：**
    *   **核心思想：** 在推理阶段，根据用户的问题动态筛选最相关的视频帧。
    *   **分段局部注意力（Segmented Local Attention）：** 为解决长文本中 LLM 常见的“迷失在中间”（Lost in the Middle）位置偏差问题，将长视频切分为短片段，在片段内计算问题 Token 与视频 Token 的注意力分数，以此作为帧的重要性评分，保留 Top-K 关键帧。

### 6. 主要结论与发现
*   **极致压缩可行：** 证明了通过学习，LLM 能够将一帧的信息浓缩至 1 个 Token 而不损失核心语义。
*   **性能提升：** XComp 在 LVBench 上将准确率从 42.9% 提升至 46.2%，在 2B 规模模型中达到了 SOTA 水平。
*   **正向反馈循环：** 由于 Token 效率极高，模型可以输入比基准模型多 2-4 倍的帧数，且性能随帧数增加持续上升，而传统模型在帧数过多时往往因上下文过载导致性能下降。

### 7. 优点（亮点）
*   **极高的 Token 效率：** 实现了“每帧一 Token”的极限，极大扩展了 VLM 处理超长视频的潜力。
*   **解决位置偏差：** 创新性地提出分段局部注意力来计算相关性评分，有效缓解了长序列中的注意力衰减问题。
*   **数据高效：** 仅需极少量的微调数据即可赋予现有模型强大的压缩理解能力。

### 8. 不足与局限
*   **模型规模限制：** 主要实验集中在 2B 规模模型上，虽然在 LLaVA-Next 上做了验证，但尚未在大规模（如 70B+）模型上进行全面测试。
*   **任务覆盖：** 虽然在长视频 QA 上表现优异，但在需要极高空间分辨率的任务（如细粒度 OCR 或微小物体检测）中，极致压缩可能会导致空间细节丢失。
*   **联合优化：** 目前 QC-Comp 主要在推理阶段使用，尚未实现帧筛选与 Token 压缩的端到端联合训练。
