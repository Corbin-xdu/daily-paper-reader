## 摘要
超高分辨率（UHR）遥感图像将公里级的上下文信息与可能仅占几个像素的查询关键证据结合在一起。这种巨大的空间尺度导致视觉 Token 呈二次方爆炸，并阻碍了从小目标中提取信息。以往的工作采用直接下采样、密集切片或全局 top-k 剪枝，这些方法要么损害了查询关键的图像细节，要么产生了不可预测的计算开销。在本文中，我们提出了 UHR-BAT，这是一个查询引导且区域忠实的 Token 压缩框架，旨在严格的上下文预算下高效选择视觉 Token。具体而言，我们利用文本引导的多尺度视觉 Token 重要性估计，有效地解决了实现精确且低成本特征提取的挑战。此外，通过引入区域级的保留与合并策略，我们减轻了视觉 Token 的冗余，进一步降低了计算预算。实验结果表明，UHR-BAT 在各种基准测试中均达到了最先进的性能。代码可在 https://github.com/Yunkaidang/UHR 获取。

## Abstract
Ultra-high-resolution (UHR) remote sensing imagery couples kilometer-scale context with query-critical evidence that may occupy only a few pixels. Such vast spatial scale leads to a quadratic explosion of visual tokens and hinders the extraction of information from small objects. Previous works utilize direct downsampling, dense tiling, or global top-k pruning, which either compromise query-critical image details or incur unpredictable compute. In this paper, we propose UHR-BAT, a query-guided and region-faithful token compression framework to efficiently select visual tokens under a strict context budget. Specifically, we leverage text-guided, multi-scale importance estimation for visual tokens, effectively tackling the challenge of achieving precise yet low-cost feature extraction. Furthermore, by introducing region-wise preserve and merge strategies, we mitigate visual token redundancy, further driving down the computational budget. Experimental results show that UHR-BAT achieves state-of-the-art performance across various benchmarks. Code will be available at https://github.com/Yunkaidang/UHR.

UHR-BAT 提出了一个**查询引导（Query-guided）**且**区域忠实（Region-faithful）**的压缩框架，核心流程如下：
*   **多尺度输入与尺度特定编码**：构建多个分辨率的图像流（如从 672 到 4032），引入“尺度特定位置编码”和“尺度嵌入”，确保不同分辨率的 Token 在空间上对齐且可区分。

*   **区域级保留与合并（RPM）策略**：
    *   **区域划分**：利用 K-means（结合特征和坐标）或 SAM 模型将 Token 划分为语义一致的区域。
    *   **保留与合并**：在每个区域内，保留重要性高于平均值的 Token，而将剩余的冗余 Token 聚合成一个代表性 Token（Mean Pooling）。
    *   **优先级排序**：根据区域平均得分进行序列化，确保在达到硬性 Token 预算（Budget）时，最关键的区域和细节被优先保留。

### 6. 主要结论与发现
*   **性能领先**：UHR-BAT 在 XLRS-Bench 上取得了 44.0% 的加权平均准确率，超越了包括 GPT-4o 和 InternVL3-78B 在内的众多强力基准。
*   **预算高效**：通过 RPM 策略，模型在极高的压缩比（如 32.83×，仅保留 4k Token）下仍能保持极高的推理精度。

### 7. 优点（亮点）
*   **预算感知（Budget-Aware）**：首次在遥感领域强调了在“硬性 Token 预算”下进行压缩，这对于长文本 LLM 的显存管理至关重要。
*   **解耦分辨率与成本**：成功将输入图像的高分辨率与计算成本解耦，使得在消费级显卡上处理超大遥感图像成为可能。
*   **区域忠实性**：通过区域内合并而非简单全局剔除，有效防止了稀疏小目标（如海上的孤立小船）被作为噪声剪枝掉。

### 8. 不足与局限

*   **应用限制**：主要针对静态图像理解，对于需要极高实时性的动态视频流遥感任务，其 4 秒左右的延迟可能仍显不足。
