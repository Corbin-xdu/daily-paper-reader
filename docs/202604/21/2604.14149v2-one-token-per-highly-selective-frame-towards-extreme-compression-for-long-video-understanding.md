## 摘要
由于视频帧数庞大，长视频理解对视觉语言模型（VLM）而言具有内在挑战性。由于每个视频帧通常会扩展为数十或数百个 token，大语言模型（LLM）有限的上下文长度迫使 VLM 只能稀疏地感知视频帧，从而丢失了时间信息。为了解决这一问题，我们探索了极致的视频 token 压缩，旨在 LLM 的最后一层实现每帧仅对应一个 token。我们的核心见解是，先前方法广泛采用的基于启发式的压缩容易导致信息丢失，因此有必要将 LLM 层监督为可学习且渐进式的 token 级压缩模块（LP-Comp）。这种压缩使我们的 VLM 能够处理 2 到 4 倍的视频帧，并提升性能。为了进一步提高 token 效率，我们研究了帧级压缩，即通过 LLM 层的内部注意力分数选择与查询最相关的帧，称之为问题条件压缩（QC-Comp）。与以往研究的一个显著区别在于，我们通过将长视频分割成短片段并采用局部注意力，缓解了长上下文中 LLM 注意力的位置偏差（即过度集中在序列的开头和结尾）。总之，我们结合了 token 级和帧级压缩，构建了一个名为 XComp 的长视频理解极致压缩模型，实现了显著更高的压缩率并支持更密集的帧采样。我们的 XComp 基于 VideoChat-Flash 进行微调，采用了一种高效的数据监督压缩微调阶段，仅需 2.5% 的监督微调数据，便将 LVBench 上的准确率从 42.9% 提升至 46.2%，并增强了多个其他长视频基准测试的表现。

## Abstract
Long video understanding is inherently challenging for vision-language models (VLMs) because of the extensive number of frames. With each video frame typically expanding into tens or hundreds of tokens, the limited context length of large language models (LLMs) forces the VLMs to perceive the frames sparsely and lose temporal information. To address this, we explore extreme video token compression towards one token per frame at the final LLM layer. Our key insight is that heuristic-based compression, widely adopted by previous methods, is prone to information loss, and this necessitates supervising LLM layers into learnable and progressive modules for token-level compression (LP-Comp). Such compression enables our VLM to digest 2x-4x more frames with improved performance. To further increase the token efficiency, we investigate frame-level compression, which selects the frames most relevant to the queries via the internal attention scores of the LLM layers, named question-conditioned compression (QC-Comp). As a notable distinction from previous studies, we mitigate the position bias of LLM attention in long contexts, i.e., the over-concentration on the beginning and end of a sequence, by splitting long videos into short segments and employing local attention. Collectively, our combined token-level and frame-level leads to an extreme compression model for long video understanding, named XComp, achieving a significantly larger compression ratio and enabling denser frame sampling. Our XComp is finetuned from VideoChat-Flash with a data-efficient supervised compression tuning stage that only requires 2.5% of the supervised fine-tuning data, yet boosts the accuracy from 42.9% to 46.2% on LVBench and enhances multiple other long video benchmarks.

#### 1. 核心问题与整体含义（研究动机和背景）
长视频理解的主要瓶颈在于视觉 Token 的爆炸式增长。现有的视觉语言模型（VLM）在处理长视频时，由于每个视频帧会被编码为数十甚至数百个 Token，极易超出大语言模型（LLM）的上下文长度限制。这迫使模型不得不进行稀疏采样，从而丢失关键的时间细节。
本文的研究动机在于：现有的 Token 压缩方法多基于简单的启发式规则（如池化或固定采样），缺乏学习过程，容易导致信息丢失。作者旨在探索一种**极致压缩**方案，目标是在 LLM 的最终输出层实现“**每帧仅保留一个最具代表性的 Token**”，从而允许模型输入更密集的视频帧（提升 2-4 倍采样率）。

#### 2. 方法论：核心思想与关键技术
XComp 框架结合了 Token 级和帧级两个维度的压缩策略：

*   **LP-Comp（可学习且渐进式 Token 压缩）**：
    *   **核心思想**：不再依赖训练外的启发式规则，而是通过监督微调让 LLM 层学会如何将多 Token 压缩为单 Token。
    *   **渐进式调度**：采用余弦调度函数（Cosine Schedule），在 LLM 的 $L$ 层中平滑地减少每帧的 Token 数，避免信息在单层内骤减。
    *   **后缀保留（Suffix-Preservation）**：基于 Decoder-only LLM 的因果注意力特性，压缩时优先保留每帧序列末尾的 Token，因为这些 Token 已经通过注意力机制吸收了前序 Token 的特征。
*   **QC-Comp（问题条件帧级压缩）**：
    *   **核心思想**：根据用户提出的问题，动态筛选最相关的视频帧。
    *   **分段局部注意力（Segmented Local Attention）**：为解决长序列中 LLM 常见的“位置偏差”（Lost in the Middle，即注意力集中在首尾）问题，将长视频切分为短片段，在片段内计算问题 Token 与视频 Token 的注意力分数，以此作为帧的重要性评分。

#### 6. 主要结论与发现
*   **极致压缩可行性**：通过监督微调，LLM 完全有能力将一帧的信息浓缩至单个 Token 而不损失核心语义。
*   **性能随帧数正相关**：XComp 展现出独特的趋势——随着输入帧数的增加，性能持续提升，而传统模型在帧数过多时往往因上下文过长导致性能下降。
*   **位置偏差缓解**：分段局部注意力能有效识别视频中间部分的逻辑关键帧，显著优于全局注意力筛选。

#### 7. 优点（亮点）
*   **极高的数据效率**：仅需极少量的微调数据即可赋予模型强大的压缩能力。
*   **架构兼容性强**：LP-Comp 是一种通用的层内压缩方案，可轻易移植到其他基于 Transformer 的 VLM 中。
*   **推理成本大幅降低**：在保持甚至提升长视频理解准确率的同时，显著降低了计算开销和显存占用。

#### 8. 不足与局限
*   **细粒度感知损失**：在需要极高空间分辨率的任务（如 VideoOCR）中，由于 Token 压缩过于剧烈，性能较原始模型有轻微下降。
*   **任务类型局限**：目前主要在多选题（Multiple-choice）形式的 Benchmark 上表现优异，对于生成式长视频描述或复杂指令遵循的评估相对较少。
*   **训练阶段未整合 QC-Comp**：目前的帧级筛选仅在推理阶段使用，若能在训练阶段实现端到端优化，可能会有更高上限。
