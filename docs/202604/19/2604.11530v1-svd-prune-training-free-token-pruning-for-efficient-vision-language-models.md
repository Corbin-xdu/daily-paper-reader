## 摘要
视觉语言模型（VLM）通过联合处理视觉和文本信息，彻底改变了多模态学习。然而，由于处理长序列视觉 Token 对计算和内存的高需求，它们面临着重大挑战。许多现有方法依赖于局部启发式策略，例如注意力分数或 Token 范数。然而，这些标准存在位置偏差和信息分散的问题，限制了它们在高剪枝率下保留核心内容的能力，并导致在视觉细节丰富的图像上性能下降。为了解决这些问题，我们提出了 SVD-Prune，一种基于奇异值分解（SVD）的无需训练、即插即用的 Token 剪枝方法。该方法对视觉 Token 特征矩阵进行分解，并利用统计杠杆得分选择前 K 个 Token，确保仅保留对主导全局方差贡献最大的 Token。实验表明，在极端的视觉 Token 预算下，SVD-Prune 始终优于先前的剪枝方法，即使在仅剩 32 和 16 个视觉 Token 的情况下也能保持强劲的性能。

## Abstract
Vision-Language Models (VLM) have revolutionized multimodal learning by jointly processing visual and textual information. Yet, they face significant challenges due to the high computational and memory demands of processing long sequences of vision tokens. Many existing methods rely on local heuristics, such as attention scores or token norms. However, these criteria suffer from positional bias and information dispersion, limiting their ability to preserve essential content at high pruning ratios and leading to performance degradation on visually detailed images. To address these issues, we propose SVD-Prune, a trainingfree, plug-and-play token pruning method based on Singular Value Decomposition. It decomposes the vision token feature matrix and selects the top-K tokens using statistical leverage scores, ensuring only tokens contributing most to the dominant global variance are preserved. Experiments show that SVD-Prune consistently outperforms prior pruning methods under extreme vision token budgets, maintaining strong performance even with 32 and 16 vision tokens.

### 1. 核心问题与整体含义（研究动机和背景）
视觉语言模型（VLM）在处理图像时，通常会将图像分解为大量的视觉标记（Vision Tokens），这导致了巨大的计算开销和内存占用。现有的视觉标记剪枝方法主要依赖局部启发式策略（如注意力分数、Token 范数或跨模态相似度），但这些方法存在两个核心缺陷：
*   **位置偏差（Positional Bias）：** 由于大语言模型（LLM）解码器使用因果掩码（Causal Masking），靠后的 Token 往往因为可关注的后续对象较少而获得较低的注意力分数，导致剪枝决策受位置而非语义驱动。
*   **信息分散：** 局部指标难以捕捉全局视觉结构，在极端剪枝率（如保留少于 64 个 Token）下，模型性能会剧烈下降。
**研究动机：** 提出一种无需训练、基于全局特征结构的剪枝方法，以在极低 Token 预算下保持 VLM 的推理能力。

### 2. 论文提出的方法论：SVD-Prune
该方法的核心思想是利用**奇异值分解（SVD）**从全局方差的角度识别最重要的视觉标记。其流程分为四个阶段：
1.  **全局模式提取：** 对视觉编码器输出的特征矩阵 $F$ 进行 SVD 分解（$F = U\Sigma V^\top$）。其中左奇异向量 $U$ 代表 Token 空间的特征方向，奇异值 $\Sigma$ 代表各方向的方差强度。
2.  **主导方差截断：** 计算累积解释方差比例，选择最小的 $k$ 个奇异值，使得它们保留了原始特征中 $\epsilon$（超参数，通常在 0.7 到 0.95 之间）的方差。这能有效过滤噪声并保留核心视觉信号。
3.  **计算杠杆得分（Leverage Scores）：** 利用截断后的 $U$ 矩阵计算每个 Token 的统计杠杆得分。得分越高，说明该 Token 对全局主导模式的贡献越大。
4.  **Token 选择与恢复：** 根据得分保留前 $m$ 个 Token，并将其重新按原始空间顺序排列，以保持位置编码的连续性，随后输入 LLM 解码器。
