## 摘要
在这项工作中，我们开发并测试了三种基于向量量化（VQ）的模型权重压缩技术。为了缓解码本崩溃并实现端到端训练，我们采用了基于余弦相似度的分配方式。借鉴可微分 K-Means（DKM）中基于注意力机制的公式，我们通过结合余弦相似度分配、top-1 采样和直通估计器进一步改进了该方法，从而消除了对加权平均重构的需求。最后，我们研究了利用可微分神经架构搜索（NAS）来自适应选择逐层量化配置，从而进一步优化压缩过程。尽管我们的方法在所有量化水平上并不总是优于现有方法，但它为基于 VQ 的模型压缩方法的设计权衡和行为提供了有益的见解。

## Abstract
In this work, we developed and tested 3 techniques for vector quantization (VQ) based model weight compression. To mitigate codebook collapse and enable end-to-end training, we adopted cosine similarity-based assignment. Building on ideas from attention-based formulations in Differentiable K-Means (DKM), we further improved this approach by using cosine similarity for assignment combined with top-1 sampling and a straight-through estimator, thereby eliminating the need for weighted-average reconstruction. Finally, we investigated the use of differentiable neural architecture search (NAS) to adaptively select layer-wise quantization configurations, further optimizing the compression process. Although our method does not consistently outperform existing approaches across all quantization levels, it provides useful insights into the design trade-offs and behaviors of VQ-based model compression methods.

### 1. 核心问题与整体含义（研究动机和背景）
该研究聚焦于深度神经网络的**权重压缩**，特别是**向量量化（Vector Quantization, VQ）**技术。
*   **背景**：VQ 相比传统的标量量化具有更高的压缩潜力，且非常契合存内计算（CIM）架构。
*   **核心问题**：
    1.  **码本崩溃（Codebook Collapse）**：训练过程中部分码本条目从未被分配到权重，导致资源浪费。
    2.  **训练与推理不一致**：现有的 SOTA 方法（如 DKM）在训练时使用“软分配”（加权平均），而推理时使用“硬分配”（最近邻），这种不匹配需要复杂的温度参数调优。
    3.  **配置灵活性不足**：缺乏一种能够自动在不同层之间平衡线性量化（LQ）和向量量化（VQ）的机制。

### 2. 方法论：核心思想与关键技术
论文提出了三种改进技术来实现高效的量化感知训练（QAT）：

*   **基于余弦相似度的分配与投影缩放（STE）**：
    *   **核心发现**：预训练权重的“方向”分布比“模长”更均匀。
    *   **技术细节**：使用余弦相似度进行码本分配以缓解码本崩溃。为了补偿丢失的模长信息，为每个权重向量引入一个额外的标量缩放因子 $s$。通过直通估计器（STE）将 $s$ 视为常数进行梯度回传。
*   **可微分硬注意力机制（Hard Attention VQ-QAT）**：
    *   **核心思想**：将 VQ 建模为注意力操作。权重向量作为 Query，归一化码本作为 Key，码本条目作为 Value。
    *   **流程**：前向传播采用 **Top-1 采样**（One-hot 分配），确保训练与推理完全一致；后向传播利用 STE 将梯度直接传回概率分布，从而实现码本的端到端优化，无需 DKM 中的迭代聚类。
*   **基于 ProxylessNAS 的混合量化选择**：
    *   **核心思想**：引入神经架构搜索（NAS）来自动决定每一层使用 VQ 还是 LQ。
    *   **技术细节**：为每层设置两个并行分支（VQ 和 LQ），通过可学习的架构参数进行概率采样。在损失函数中加入存储开销惩罚项，以在精度和压缩率之间取得平衡。
